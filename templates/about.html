About CineMatch AI

CineMatch AI is a demonstration of a Content-Based Filtering recommendation system built with Flask and Scikit-Learn.

How it Works

  • Data Analysis: We analyze movie genres and descriptions using TF-IDF (Term Frequency-Inverse Document Frequency) to create a mathematical vector for each movie.
  • User Profiling: Based on your ratings, we build a weighted user profile vector. Positive ratings pull the profile towards the movie's features, while negative ratings push it away.
  • Similarity Search: We use Cosine Similarity to compare your profile against thousands of movies to find the closest matches in our multi-dimensional vector space.

Tech Stack

  • Python
  • Flask
  • Scikit-Learn
  • Vanilla JS
